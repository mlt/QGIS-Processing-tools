Channel migration tool
  •  
  •  

Channel migration tool assess annual erosion rate by application of Dynamic Time Warping method (as implemented in R dtw package) to aligned signed curvature of a channel. Smooth spline is used to derive curvature as a parametric function of position along the channel (from 0 to 1).

This is essentially an alternative to

  • Planform Statistics tool
  • …

Information

This method uses DTW that matches peaks in curvature and it does not rely on spatial proximity. Thus erosion rates might be overestimated comparing to other tools.
